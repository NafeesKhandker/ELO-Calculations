Objective

Calculate each chess player’s expected score using the ELO rating system based on the provided dataset, compare it with their actual score, and identify the top five players who over-performed and under-performed relative to expectations.

Approach

Dataset description

For this assignment, I will use the output dataset generated from Project 1, which is a preprocessed CSV file containing structured chess tournament results. The dataset can be accessed here: https://raw.githubusercontent.com/NafeesKhandker/ELO-Calculations/refs/heads/main/Chess_result_data.csv

This dataset is already cleaned and organized, making it suitable for direct analysis. The dataset contains the following columns:

  • pair_num: Unique pairing number assigned to each player

  • Player_name: Name of the player

  • State: Player’s stat

  • Total_points: Total points scored by the player in the tournament

  • Rating: Player’s pre-tournament rating

  • Avg_opponent_rating: Average rating of all opponents faced by the player

This structured dataset simplifies the calculation of expected scores and performance differences using the ELO rating system.

Anticipated Challenges

The main anticipated challenge for this project is implementing the ELO expected score calculation correctly and efficiently. Since each player competes against multiple opponents, the expected score must be calculated individually for each match and then aggregated. Another challenge is ensuring accurate numerical computation, especially when dealing with floating-point precision during exponential calculations.

In this step, we are calculating the expected score per game for a player. The expected score for Player A against Player B (average opponents rating in our case) is given by:

\[ E_A = \frac{1}{1 + 10^{\frac{R_B - R_A}{400}}} \]

Where:

  • \(E_A\) = Expected score of Player A
  • \(R_A\) = Rating of Player A
  • \(R_B\) = Rating of Player B

The second Elo formula updates a player’s rating after each game based on the actual and expected scores:

\[ R_{\text{new}} = R_{\text{old}} + K \times (S - E) \]

Where:

  • \(R_old\) = current rating of the player
  • \(R_new\) = updated rating after the game
  • \(K\) = K-factor (adjustment coefficient, e.g., 20, 32)
  • \(S\) = actual score of the game (1 = win, 0.5 = draw, 0 = loss)
  • \(E\) = expected score, calculated as:
